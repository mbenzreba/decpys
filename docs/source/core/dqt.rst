Alignment
---------

.. py:class:: Alignment

    Enumerates alignments that can be used in widgets' ``alignment``, ``setAlignment``, and other
    related methods.

    Enumerated values:

    * ALIGN_LEFT
    * ALIGN_RIGHT
    * ALIGN_HCENTER
    * ALIGN_JUSTIFY
    * ALIGN_TOP
    * ALIGN_BOTTOM
    * ALIGN_VCENTER
    * ALIGN_BASELINE
    * ALIGN_CENTER
    * ALIGN_NONE
